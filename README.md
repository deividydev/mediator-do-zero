# mediator-do-zero
## Descrição

Este repositório contém uma implementação do padrão **Mediator** do zero, sem o uso do pacote **MediatR**, criado para demonstrar minha compreensão do padrão e habilidades em **C#** e **.NET Core**. O **Mediator** é um padrão de design comportamental que permite que os objetos se comuniquem entre si sem que precisem se referir diretamente uns aos outros, facilitando a manutenção e escalabilidade do sistema.

Essa implementação foi desenvolvida como uma solução simples e flexível para demonstrar o conceito e como ele pode ser aplicado em arquiteturas como **Clean Architecture** e **Vertical Slice Architecture**.

## Tecnologias Utilizadas

- **.NET Core** (versão 9)
- **C#**

## Funcionalidades

- Implementação do padrão **Mediator** sem o uso de bibliotecas externas.
- Suporte à criação de **Handlers** para gerenciar as requisições.
- Estrutura modular e fácil de entender, adequada para demonstrações e para ser expandida.
- Exemplo de uso com requisições simples.
